# java basic test4
There are some steps for you to run those programs:
1. 进入 JavaTest4。
2. 运行build.sh。


* 这个程序在会打印出结果后自动退出（包括清理你的容器和镜像）,如果你不想运行完之后就退出程序。可以在build.sh里面将`docker-compose up --exit-code-from java-1;sh stop.sh;`改为`docker-compose up;`。
